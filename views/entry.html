Grunt 最重要的就是 package.json 和 Gruntfile.js，首先需要在项目的根目录创建package.json 和 Gruntfile.js 文件，当然也可以通过命令 grunt-init 来创建，更多详情起跳转到 GitHub 。

package.json 的创建

在终端项目的根目录下输入： npm init 根据提示一步一步的来，即可创建完成。 pacage.json 例子：

{
  "name": "project-one",
  "version": "0.0.1",
  "author": "zhangshuhui",
  "devDependencies": {
    "grunt": "^0.4.5",
    "grunt-contrib-uglify": "^0.9.1"
  }
}
通过 npm init 生成的 pacage.json 中还有一些其它字段，不过可以精简掉。更多参数请查看官方文档。
